AMBA APB interface

Input Ports

Variables

  • logic[32-1:0] addr 
    Address bus
  • bit en_protocol_checks 
    Switch to enable protocol checks ( 1         on; 0 - off)
  • bit en_ready_low_max_time 
    Switch to enable maximum number of clock cycles during when a signal can be kept low ( 1         on; 0 - off)
  • bit en_rst_checks 
    Switch to enable reset checks ( 1         on; 0 - off)
  • bit en_x_z_checks 
    Switch to enable X/Z checks ( 1         on; 0 - off)
  • logic enable 
    Enable signal
  • bit has_error_signal 
    Switch which indicates that the agent uses error signal ( 1         on; 0 - off)

    Protection bus
  • logic[32-1:0] rdata 
    Read data bus
  • logic ready 
    Ready signal
  • logic reset_n 
    Reset signal
  • logic[2-1:0] sel 
    Select signal
  • logic slverr 
    Error signal
  • logic[4:0] strb 
    Write strobe bus
  • logic[32-1:0] wdata 
    Write data bus
  • logic write 
    Direction signal
